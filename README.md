# Docx文件验证工具

## 简介

这是一个基于C#开发的Docx文件验证工具，使用微软官方的DocumentFormat.OpenXml库来验证.docx文件是否符合微软官方标准。

当您无法找到或使用微软官方提供的验证工具时，可以使用本工具快速验证Docx文件的格式正确性。

## 功能特点

- 自动扫描指定目录下的所有.docx文件
- 使用DocumentFormat.OpenXml库进行验证
- 检查文档结构的完整性
- 提供详细的验证结果和错误信息
- 简单易用的控制台界面

## 环境要求

- .NET 8.0 或更高版本
- DocumentFormat.OpenXml 3.4.1 或更高版本

## 使用方法

1. 将需要验证的.docx文件放入项目根目录下的`Docx`文件夹中
2. 运行程序：
   ```bash
   dotnet run
   ```
3. 程序会自动扫描并验证所有.docx文件
4. 查看验证结果，按任意键退出程序

## 验证内容

本工具会验证以下内容：

- 文件是否为有效的Word文档格式
- 主文档部分是否存在
- 文档内容结构是否完整
- 文档正文部分是否存在

## 验证结果

- **有效**：文件符合微软Docx标准，结构完整
- **无效**：文件存在格式问题或结构不完整，程序会显示具体的错误信息

## 技术说明

本工具使用微软官方的DocumentFormat.OpenXml库，该库是微软提供的用于处理Office Open XML格式的标准库。通过该库，我们可以：

- 正确解析.docx文件的内部结构
- 验证文件是否符合Office Open XML规范
- 检测常见的格式错误和结构问题

## 注意事项

- 本工具仅验证文件结构是否符合标准，不验证文档内容的正确性
- 验证过程中不会修改原始文件
- 建议在验证前备份重要文件
